## 💻 Setup di Berbagai Aplikasi IPTV

### 1️⃣ **VLC Media Player** (Windows, Mac, Linux)

**Cara Manual:**
1. Buka VLC
2. Menu → **Media** → **Open Network Stream...**
3. Paste link: `https://raw.githubusercontent.com/ziizidavid/VSA/main/IPtv`
4. Klik **Play**

**Cara Playlist File:**
1. Buat file `iptv.m3u` (text editor)
2. Isi dengan: `#EXTM3U` (baris 1) dan link di bawahnya
3. Drag & drop ke VLC atau buka melalui File → Open

## ⚙️ Troubleshooting

### ❌ Playlist tidak load
- **Solusi 1**: Refresh/reload playlist di aplikasi
- **Solusi 2**: Clear cache aplikasi IPTV
- **Solusi 3**: Gunakan VPN jika ada pemblokiran geografis

### ❌ Channel tidak keluar
- **Solusi**: Update playlist (biasanya auto-update setiap jam)
- Cek: Channel mungkin sedang offline/maintenance

### ❌ Buffering / Loading lambat
- **Solusi**: Gunakan VPN server yang lebih cepat
- Atau gunakan aplikasi IPTV player yang lebih ringan

### ❌ Audio/Video corrupt
- **Solusi**: Refresh stream atau pilih channel lain
- Setiap channel bisa di-play dengan kualitas berbeda

## 💡 Tips & Trik

### 1. **Gunakan Multiple Sources**
- Tambahkan playlist lain sebagai backup
- Jika satu source down, ada yang lain

### 2. **VPN untuk Konten Geo-Restricted**
- Beberapa channel hanya available di negara tertentu
- Gunakan VPN dengan exit point di negara tersebut

### 3. **Favorite/Bookmark Channels**
- Semua aplikasi IPTV punya fitur favorites
- Organize channels favorit untuk akses cepat

### 4. **Custom EPG (Electronic Program Guide)**
- Aplikasi seperti Kodi & TiviMate support EPG external
- EPG memberi info jadwal & deskripsi acara

### 5. **Screen Recording & Cast**
- Untuk Smart TV/Firestick: Gunakan Chromecast/AirPlay
- Android: Built-in screen cast ke TV lain
